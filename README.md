# PA-MEQA

Priority-Aware Multi-Objective Q-Learning Framework for Mobile Charger Scheduling in Wireless Rechargeable Sensor Networks.

## Features

- Priority-aware reinforcement learning
- PDEO-based event importance modeling
- Adaptive reward shaping
- Time-to-failure estimation
- Coverage redundancy reduction
- Mobile charger scheduling
- Comparative evaluation against Q-Learning and MEQA

## Results

- 20.95% reduction in dead nodes
- Improved Quality of Event Detection (QED)
- Better sensing coverage and detection probability

## Technologies

Python, NumPy, Reinforcement Learning, Wireless Sensor Networks
